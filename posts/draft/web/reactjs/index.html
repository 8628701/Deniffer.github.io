React

React is a Framework that lets you divide up your website into reusable Components

  • Props: inputs passed from a parent to a child component. (Immutable)
  • State: Private information maintained By a component (Mutable)
  • Each component is a function with props as the input, and returns HTML like code.

其实react 用下来感觉就是分而治之的感觉，利用组件化的思想把业务拆分成更小的模块 然后实现。 印象比较深刻的点在于，react 的数据流动，在之前我在写客户端的代码的时候，更多是以一种双向数据流动的方式在走，你可以随时修改数据的值，无论你在哪一个模块。但是react更强调的单向数据流动，数据只能从父组件流向子组件，子组件不能修改父组件的数据。只能通过全局变量的方法来修改值，然后父组件去监听这个值做render，感觉用起来还是挺怪异的。

还有一个点比较印象深刻的在于，react的function component 强调了 pure function的概念，把所有的副作用都独立到useEffect这个函数里面，这点做的还挺好的，因为component的刷新是很容易就会发生的，但是数据的获取并不一定是呀跟着component的刷新一起走。 这样做的好处在于可以很好的解决无论component刷新多少次，都和预期想要的效果是一致的。
